LaraAdmin Panel is a **self-hosted, all-in-one business management platform** that bundles 40+ modules into a seamless administration interface. From file management and backups to LDAP authentication and license activation servers — everything is accessible from a single, unified dashboard.

---

## ✨ Highlights

| ⚡ Capability | 🛠️ What it does |
|---|---|
| **System Admin** | File managers, backups, logs, OPcache, terminal, Artisan GUI |
| **User Control** | Roles/permissions, LDAP/AD, social login, 2FA, impersonation |
| **Dev Tooling** | Git web UI, online code editor, module generator, settings editor |
| **Content** | Page builder, blog/posts, wiki, SEO management, custom fields |
| **Business Ops** | Support tickets (IMAP), events, time-off, password vault, license server |
| **Customer** | Customer portal, frontend section |

## 🚀 Installation

```bash
# 1. Clone the repository
git clone https://github.com/your-username/laraadmin-panel.git
cd laraadmin-panel

# 2. Install PHP dependencies
composer install --no-dev --optimize-autoloader

# 3. Install & build frontend assets
npm install
npm run build

# 4. Environment setup
cp .env.example .env
php artisan key:generate

# 5. Configure .env with database & services, then run:
php artisan migrate

# 6. Storage link
php artisan storage:link

# 7. Cache for production
php artisan config:cache
php artisan route:cache
php artisan view:cache

# 8. Seed admin user
php artisan db:seed --class=AdminUserSeeder
```

### 🐳 Quick Start with Docker (Laravel Sail)

```bash
cp .env.example .env
php artisan sail:install
./vendor/bin/sail up -d
./vendor/bin/sail artisan migrate
./vendor/bin/sail artisan storage:link
```
